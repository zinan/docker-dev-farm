# Docker Developer Farm

All in one development stack for PHP developers

### Requirements
* Docker
* Docker Compose

### Setup
Just clone or download repo and run **install.sh**

If you need more tuning, please change default values in **.env** file


### Supported Images
- PHP 5.6
- PHP 7.1
- PHP 7.2
- PHP 7.3
- Nginx
- Apache
- Nodejs
- Redis
- Mongodb
- Mysql
- Redis Commander
- PhpMyAdmin
- PostgreSql
